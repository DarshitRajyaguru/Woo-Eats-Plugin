WooEats plugin to manage online pickup and delivery with custom addons.

== Description ==

WooEats allows you to manage pickup and delivery options for your WooCommerce store, along with custom product addons.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wooeats` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the WooEats settings page to configure the plugin.

== Frequently Asked Questions ==

= How do I add custom product addons? =
Go to the product edit page in WooCommerce and use the WooEats Addons meta box.
